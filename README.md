# The Brass Door Bar & Restaurant

A stunning, atmospheric website for The Brass Door Bar & Restaurant in Carlingford.

## Features

- Dark, sophisticated design with brass accents
- Smooth animations and hover effects using Framer Motion
- Signature cocktails showcase
- Live music and atmosphere sections
- Fully responsive design
- Built with modern web technologies

## Tech Stack

- Vite
- React 18
- Tailwind CSS
- Framer Motion

## Getting Started

### Install dependencies:
```bash
npm install
```

### Run development server:
```bash
npm run dev
```

### Build for production:
```bash
npm run build
```

### Preview production build:
```bash
npm run preview
```

## Design Inspiration

Inspired by White's Tavern Belfast and Ben Madigans Bar, this site captures the essence of:
- Attention to detail
- Passion for hospitality
- Beautiful cocktails
- Live music atmosphere
- Warm, welcoming ambiance

## Color Palette

- Background: Deep blacks and charcoals (#0a0a0a, #18181b)
- Accent: Brass/Gold (#D4AF37, #B8860B)
- Text: Zinc grays and whites

## Typography

- Headings: Cormorant Garamond (Serif)
- Body: Inter (Sans-serif)
